Manipulación del DOM con JQuery

JQuery viene con algunos métodos preconfigurados que facilitan enormemente el uso del DOM:

    GET
  • text() - retorna el contenido de un elemento como texto
  • html() - retorna el innerHTML de un elemento etiquetas HTML incluídas
  • val() - retorna el valor de los campos input

Quitar/Añadir elementos

Añadir elementos nuevos

  • append() - Inserta el contenido al final de los elementos seleccionados
  • preprend() - Inserta el contenido a inicio de los elementos
  • after() - Inserta el contenido después del elemento seleccionado
  • before() - Inserta el contenido antes del elemento seleccionado
